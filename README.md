### Notes to myself because I often forgot

To Start mongodb (ubuntu): 
```
sudo systemctl start mongod
```
To Access mongodb (ubuntu):
```
mongo 
```
To access mongodb with credentials (ubuntu): 
```
mongo -u username -p password
```
